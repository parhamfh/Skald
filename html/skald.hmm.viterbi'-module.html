transition_max_k(j, T1, b, B, T, observed, p)

source code 

Returns the next state which maximizes probability of transition to that state and emission probabilities. Returns a tuplet with index of state as well as value of probability.

Parameters:
  • j (int) - the time for which the emission probability is calculated. Is used to index T1.
  • T1 (numpy.matrix) - 2D matrix; where first index, i, is state index and second, j, is time of emission. Elements give probability of output j being emitted by state i.
  • b (BeatState.) - The state which you are currently in.
  • B (list) - the set of the 32 hidden states (the beats)
  • T (numpy.matrix) - transition matrix for hidden states
  • observed (list) - set of observed syllables
  • p (numpy.matrix) - the duration emission probability for each state
Returns: tuples
tuplet with two values, first being max probability and second being index of state that gives max probability.
